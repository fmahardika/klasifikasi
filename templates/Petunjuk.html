Web Identifikasi Awan

Sistem mengidentifikasi jenis awan altostratus, cirrus, cumulus, dan stratus


Petunjuk Penggunaan

1. Pengguna dapat memulai pengujian dengan memasukkan file gambar awan yakni dengan menekan tombol “choose file” pada menu upload file. Menu ini yang terdapat pada beranda atau dapat menekan logo web identifikasi awan untuk akses pada menu upload file.



2. Pengguna harap memastikan ekstensi file gambar atau citra yang akan dimasukkan sesuai dengan ekstensi file (.png, .jpg, dan .jpeg) yang dapat diproses oleh sistem identifikasi awan. Apabila ekstensi file telah sesuai maka pengguna dapat menekan tombol “submit” untuk melihat hasil identifikasi.



3. Pengguna kemudian memasukkan gambar atau file dan menunggu beberapa saat sampai sistem berhasil menampilkan hasil identifikasi awan dan confidence score dari gambar yang telah dimasukkan. Confidence score tertinggi dan ditampilkan teratas menjadi jenis awan yang teridentifikasi oleh sistem.
